# Resume-Screening-System-project
The Resume Screening System is a machine learning-based application that automates the process of shortlisting candidates from resumes. It uses Natural Language Processing (NLP) techniques such as tokenization, stop-word removal, and TF-IDF vectorization to convert resume text into structured data.

A Naive Bayes classifier is then applied to classify resumes into different job categories based on their content. The system helps recruiters quickly identify suitable candidates, reducing manual effort, saving time, and minimizing bias in the hiring process.

Key Benefit: Fast, accurate, and automated resume classification for efficient recruitment.
