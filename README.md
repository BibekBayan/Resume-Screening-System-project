# Resume-Screening-System
📄🤖 Resume Screening System (NLP + Naive Bayes)

The Resume Screening System is an intelligent application 💡 that automates candidate shortlisting 📑➡️✅. Traditional screening is slow ⏳ and biased ❌, so this system uses NLP 🧠 and Naive Bayes 📊 to analyze and categorize resumes efficiently.

Resumes are processed using NLP techniques like tokenization ✂️, stop-word removal 🚫, stemming/lemmatization 🔄, and TF-IDF 📈, converting unstructured text into structured data for machine learning.

A Naive Bayes classifier 🎯 is trained on labeled data to classify resumes into job roles 👨‍💻 (e.g., Data Scientist, Software Engineer). It predicts the most suitable category using probability principles 📊.

Recruiters can upload resumes 📤, and the system automatically classifies 📂 and ranks candidates ⭐ based on relevance, saving time ⏱️ and improving accuracy 🎯.

✨ Key Features
🤖 Automated resume classification
🧠 NLP-based text preprocessing
📈 TF-IDF feature extraction
⚡ Fast prediction using Naive Bayes
🖥️ Easy-to-use interface
🎯 Reduced bias & improved efficiency

🌐 Job portals
📊 Talent acquisition platforms
✅ Conclusion

This project combines ML 🤖 + NLP 🧠 to streamline recruitment, making the hiring process faster ⚡, scalable 📈, and more accurate 🎯.
